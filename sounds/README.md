# Windows 95 Sound Effects

These are base64-encoded Windows 95 system sounds for authentic retro experience.

## Sounds included:
- `startup.mp3` - Windows 95 startup sound
- `click.mp3` - Button click
- `error.mp3` - Error dialog
- `tada.mp3` - Success/achievement sound

## Usage:
Sounds are embedded as data URLs in the HTML to avoid external dependencies.
